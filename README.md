# Memorial Photo AI (한국형 영정사진 생성기)

**"누구나 존엄하게 준비하는 가장 아름다운 마지막 사진"**

이 프로젝트는 누구나 쉽고, 안전하게 고품질의 영정사진을 미리 준비할 수 있도록 돕는 AI 서비스입니다.
복잡한 스튜디오 예약 없이, 가지고 있는 사진 한 장으로 품격 있는 마지막을 준비하세요.

---

## 🌟 핵심 가치 (Core Values)

### 1. 인물 유지 & 압도적 화질 (Fal.ai InstantID)
기존 AI 서비스들의 얼굴 왜곡 문제를 해결했습니다.
- **실제 얼굴 그대로**: 사용자의 얼굴 특징을 유지한 채 80대 모습으로 자연스럽게 변환합니다.
- **초고속 생성**: "Lightning" 인프라를 통해 단 몇 초 만에 고화질의 결과를 얻을 수 있습니다.
- **한국형 디테일**: 정중한 표정과 조명, 전통적/현대적 상복 스타일을 완성도 있게 구현했습니다.

### 2. 철저한 프라이버시 (Zero-Retention)
가장 민감할 수 있는 사진 데이터, 서버에 **아무것도 남기지 않습니다.**
- **즉시 파기**: 업로드된 원본 사진은 AI 변환 즉시 영구 삭제됩니다.
- **기록 없음**: 생성된 결과물은 파일로 저장되지 않고, 오직 사용자 브라우저에서만 잠시 확인 가능합니다.

## 🛠️ 개발 여정 (Development Journey)

이 서비스가 어떻게 완성되어가고 있는지 기록합니다.

### 📍 [PR5] 인물 유지 혁신 (Identity Milestone) - **Current**
**"얼굴을 지키면서, 세월을 더하다"**
- **Fal.ai 전격 교체**: Replicate의 한계를 넘어 `fal-ai/instantid` 엔진으로 마이그레이션했습니다.
- **인물 고정(Identity Preservation)**: 이제 단순한 노인 생성이 아닌, 사용자의 실제 얼굴 특징이 반영된 영정사진이 생성됩니다.
- **성능 최적화**: "Lightning" 인프라 도입으로 생성 시간을 3~5초대로 획기적으로 단축했습니다.

### 📍 [PR4] 지능형 UX & 검증 (Intelligent UX)
**"사용자의 실수를 방지하고 신뢰를 주다"**
- **실시간 얼굴 감지**: `face-api.js`를 탑재하여 사진 업로드 시 얼굴 포함 여부를 브라우저에서 즉시 검증합니다.
- **비교 UI 적용**: 원본과 생성 결과를 한눈에 비교할 수 있는 대화형 인터페이스를 구축했습니다.

### 📍 [PR3] AI 엔진 & 프라이버시 (AI Base)
**"가장 선명하고, 가장 안전하게"**
- **초고해상도 엔진**: Replicate FLUX 엔진을 통해 스튜디오 품질의 선명도를 확보했습니다.
- **Zero-Retention**: 서버에 어떤 사진 데이터도 남기지 않는 개인정보 보호 시스템을 완비했습니다.

### 📍 [PR2] 서비스 경험 설계 (Prototype)
**"정중한 분위기를 설계하다"**
- **영정 사진 프레임**: 한국 장례식 예법에 맞는 검은 리본과 목재 프레임 테두리를 적용했습니다.
- **Mock AI 흐름**: 실제 AI 연동 전, 대기 시간과 결과 확인 프로세스를 사용자 관점에서 선제적으로 정립했습니다.

### 📍 [PR1] 기초 시스템 (Scaffold)
**"시작하다"**
- **드래그 앤 드롭**: 사용자가 직관적으로 사진을 올릴 수 있는 업로드 인터페이스를 구축했습니다.
- **파일 전송 API**: Node.js(Express) 기반의 안정적인 서버 백엔드를 구축했습니다.

---

## 💻 Tech Stack (기술 스택)
- **AI Engine**: Fal.ai InstantID (SDXL based)
- **Backend**: Node.js v20 (Express)
- **Frontend**: Vanilla JS / CSS / Face-api.js (Local detection)
